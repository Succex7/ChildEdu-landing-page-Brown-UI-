# ChildEdu Landing Page

ChildEdu is a simple, responsive landing page built with **HTML and CSS**.  
This project was created as a hands-on exercise to **master CSS Flexbox layout techniques** while building a realistic, real-world UI.

## Project Overview

The focus of this project is layout design using Flexbox, including:
- Horizontal and vertical alignment
- Spacing and distribution of elements
- Structuring sections commonly found in landing pages
- Building clean and readable layouts without frameworks

The theme is centered around education and community support.

## Features

- Hero section with call-to-action button
- Image section laid out using Flexbox
- Clean and minimal design
- Well-structured HTML and CSS

## Technologies Used

- HTML5
- CSS3
